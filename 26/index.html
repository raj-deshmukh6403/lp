PICT B.E. Courses

Pune Institute of Computer Technology

Bachelor of Engineering
Computer Engineering Department

First Year (FE)

Engineering Mathematics I (110001)

110001

Introduction to differential calculus, integral calculus, and differential equations. Applications in engineering problems.

Engineering Physics (110002)

110002

Study of mechanics, optics, quantum physics and semiconductor physics with engineering applications.

Engineering Chemistry (110003)

110003

Fundamentals of chemistry important for engineering including polymers, electrochemistry and corrosion.

Engineering Mechanics (110004)

110004

Study of forces, moments, equilibrium, friction, centroid, and moment of inertia with engineering applications.

Basic Electrical Engineering (110005)

110005

Introduction to electrical circuits, network theorems, AC fundamentals, and electrical machines.

Programming and Problem Solving (110006)

110006

Introduction to C programming, algorithms, data structures, and problem-solving techniques.

Second Year (SE)

Data Structures and Algorithms

210241

Implementation and analysis of various data structures like arrays, linked lists, stacks, queues, trees, and graphs. Algorithm design techniques.

Object-Oriented Programming

210242

Concepts of classes, objects, inheritance, polymorphism, encapsulation, and abstraction using Java programming language.

Digital Electronics and Logic Design

210243

Study of number systems, boolean algebra, logic gates, combinational and sequential circuits, and digital logic design.

Computer Organization

210244

Architecture of computer systems, CPU organization, memory hierarchy, I/O organization, and assembly language programming.

Discrete Mathematics

210245

Study of sets, relations, functions, graph theory, propositional logic, and mathematical reasoning.

Engineering Mathematics III

207003

Linear algebra, Fourier series, partial differential equations, and complex variables with applications.

Third Year (TE)

Theory of Computation

310241

Formal languages, automata theory, Turing machines, computability, and complexity theory.

Database Management Systems

310242

Relational data model, SQL, database design, normalization, transaction management, and concurrency control.

Computer Networks

310243

Network architectures, TCP/IP protocol suite, routing algorithms, network security, and socket programming.

Operating Systems

310244

Process management, memory management, file systems, I/O systems, and protection and security in operating systems.

Design and Analysis of Algorithms

310245

Algorithm design techniques, complexity analysis, greedy algorithms, dynamic programming, and NP-completeness.

Software Engineering

310246

Software development life cycle, requirements engineering, design principles, testing strategies, and project management.

Final Year (BE)

Machine Learning

410241

Supervised and unsupervised learning, neural networks, decision trees, support vector machines, and reinforcement learning.

Information and Cyber Security

410242

Cryptography, authentication mechanisms, network security, web security, and computer forensics.

Cloud Computing

410243

Cloud infrastructure, service models, virtualization, cloud storage, and cloud security concepts.

Elective I: Internet of Things

410244A

IoT architecture, protocols, sensors, actuators, and IoT application development.

Elective II: Data Science

410244B

Data mining, statistical methods, data visualization, and big data technologies.

Project Work

410245

Industry-relevant or research-oriented project to apply knowledge and skills acquired throughout the program.

This is a simplified overview of the B.E. Computer Engineering curriculum at PICT. For detailed syllabus, please refer to the official college website.

About PICT

Pune Institute of Computer Technology

Established in 1983, Pune Institute of Computer Technology (PICT) is one of the premier engineering colleges in Pune, Maharashtra.

PICT offers undergraduate and postgraduate programs in Computer Engineering, Information Technology, and Electronics & Telecommunication Engineering.

Key Features:

  • State-of-the-art infrastructure
  • Experienced faculty members
  • Industry collaborations
  • Research opportunities
  • Active placement cell

The Computer Engineering department aims to produce technically competent engineers equipped with problem-solving skills and ethical values.
